# Install

Clone the `repo`:

``` shell
git clone https://github.com/TylerDurham/cwrap-python
```

I like to use `pipx`:

``` shell
pipx install .
```

To remove, run `pipx uninstall cwrap`.

# Local Development

Clone the `repo`:

``` shell
git clone https://github.com/TylerDurham/cwrap-python
```

Create the `python` virtual environment:

``` shell
python3 -m venv .venv
```

Source the `python` environment:

``` shell
source .venv/bin/activate
```
